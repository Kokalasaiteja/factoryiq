## Overview

FactoryIQ is a comprehensive Manufacturing Excellence dashboard built with React. It provides real-time insights into production metrics, quality control, supply chain management, and project tracking for manufacturing operations.

## Getting Started

### Create React App Commands

You can create a new React app using any of the following commands:

```bash
# Using npm
npx create-react-app new-app

# Using yarn
yarn create react-app new-app

# Using npm (specify folder name)
npm init react-app new-app
```

To create the FactoryIQ project specifically:

```bash
# Create new React app
npx create-react-app factoryiq

# Navigate to project
cd factoryiq

# Install additional dependencies
npm install react-router-dom chart.js react-chartjs-2

# Start development server
npm start
```

### Installation

```bash
# Clone the repository
git clone <repository-url>

# Navigate to project directory
cd new-app

# Install dependencies
npm install
```

### Running the Application

```bash
# Start development server
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

### Build for Production

```bash
npm run build
```
